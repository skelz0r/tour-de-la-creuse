# Sortie C - Longue

## Objectif

Habituer le corps à rouler longtemps. Tester et caler la nutrition.

## Paramètres

| | |
|---|---|
| Durée | Variable selon phase |
| Intensité | RPE 4-5/10 |
| Cadence | Naturelle |
| Fréquence | 1x/semaine (WE) |

## Progression durée

| Phase | Semaines | Durée |
|-------|----------|-------|
| Base | 1-4 | 2h → 3h |
| Force | 5-9 | 3h → 3h30 |
| Spécifique | 10-14 | Enchaînements (voir fiche) |
| Affûtage | 15-16 | 2h → 1h30 |

## Comment faire

- Allure **régulière** du début à la fin
- Même effort dans les bosses (ralentir si besoin)
- Pas de sprint, pas de segments Strava
- Terrain varié si possible

## Nutrition (CRITIQUE)

### Règle d'or
Manger **AVANT** d'avoir faim. Boire **AVANT** d'avoir soif.

### Quantités
- 1 bidon / heure minimum
- 1 apport solide toutes les 30-40 min

### Quoi manger
- Barres de céréales
- Pâtes de fruit
- Banane
- Gels (si tu supportes)
- Pain d'épices

### Planning type sortie 3h
```
0:00    Départ (bidon entamé)
0:30    Barre
1:00    Bidon 2
1:15    Pâte de fruit
1:45    Barre
2:00    Bidon 3 (ou remplir)
2:15    Banane/gel
2:45    Pâte de fruit si besoin
3:00    Arrivée
```

## Erreurs courantes

- Partir trop vite (erreur #1)
- Oublier de manger les 2 premières heures
- Accélérer quand on se sent bien à mi-parcours
- Ne pas boire parce qu'il fait frais

## Indicateurs de réussite

- Même énergie début vs fin
- Pas de fringale
- Pas de coup de mou brutal
- Envie de rouler encore un peu à l'arrivée

## Si météo mauvaise

**Ne jamais sauter la sortie longue.**

Options:
- Décaler au dimanche
- Sortir sous la pluie (équipement adapté)
- Remplacer par 2x1h30 sam+dim (moins bien mais acceptable)
