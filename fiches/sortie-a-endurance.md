# Sortie A - Endurance fondamentale

## Objectif

Construire le socle aérobie. C'est la base de tout.

## Paramètres

| | |
|---|---|
| Durée | 1h15 - 1h30 |
| Intensité | RPE 3-4/10 |
| Cadence | 85-95 rpm (naturelle) |
| Fréquence | 1x/semaine |

## Comment faire

- Allure **très confortable**
- Tu dois pouvoir parler en phrases complètes
- Pas d'accélérations, pas de "je vais voir"
- Terrain plat ou légèrement vallonné

## Erreurs courantes

- Partir trop vite
- Accélérer dans les faux-plats
- Transformer en sortie "tempo" parce qu'on se sent bien
- Vouloir suivre quelqu'un de plus rapide

## Indicateurs de réussite

- Tu finis frais, pas fatigué
- Tu aurais pu continuer 30min de plus
- Pas de courbatures le lendemain

## Nutrition

- 1 bidon suffit pour 1h15
- 1 barre/banane si > 1h
